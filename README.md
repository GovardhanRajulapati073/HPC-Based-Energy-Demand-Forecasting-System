# HPC-Based-Energy-Demand-Forecasting-System

📌 Overview

This project predicts energy consumption using High Performance Computing (HPC) techniques. It compares Serial, CPU Parallel, and GPU-based models to improve speed and performance.

🎯 Objective

Forecast energy demand from time-series data

Reduce computation time using CPU & GPU

Compare performance of different models

🛠️ Tech Stack

Python

Pandas, NumPy

Scikit-learn

XGBoost

PyTorch

Numba

⚙️ Models Used

Linear Regression (Serial)

XGBoost (CPU Parallel)

LSTM (GPU)

🚀 Key Features

Parallel sliding window using Numba

CPU and GPU acceleration

Speedup comparison

Peak load detection

Error metrics (MAE, RMSE)
